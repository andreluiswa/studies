# Projeto Website – Atividade 6

Este é um breve guia sobre um projeto web que fiz no 2° termo da faculdade: ele consistia em criar um site web sobre algum jogo, o objetivo foi voltado para a criatividade dos alunos.

---

## Páginas do Projeto

- **Home (`home.html`)**  
  Apresenta o jogo *Persona 3 Reload*, trazendo descrição, história, imagens e principais novidades do remake.

- **Biografia (`pages/biografia.html`)**  
  Detalha informações sobre **Takuya Yamaguchi**, diretor responsável pelo remake, sua trajetória e visão criativa.

- **Editora (`pages/editora.html`)**  
  Página sobre a **SEGA/Atlus**, sua importância na publicação do jogo e curiosidades sobre a história da empresa.

- **Contato (`pages/contato.html`)**  
  Fornece canais de atendimento (chat, e-mails) e informações de suporte para jogadores e interessados.

---

## Como Executar o Projeto

1. Faça o download ou clone do repositório.  
2. Abra o arquivo `home.html` em qualquer navegador moderno.  
3. Use o menu do cabeçalho para navegar entre as páginas.
